# Formio renderer

<div style={{background: '#fff3cd', padding: '2px 10px', marginBottom: '1em'}}>
**DISCLAIMER**

This is an experimental preview of the renderer provided by `@open-formulieren/formio-renderer`.
Check that package/repository for an up to date status.

</div>

The following component types are supported at the time of writing:

- `bsn`
- `date` (limited to the input group widget)
- `email`
- `fieldset` (barebones, not feature complete)
- `phoneNumber`
- `radio`
- `textfield`
